## Présentation du projet
Automatiser la gestion des règles d'un dépôt GitLab.

## Objectif du projet
Créer un outil capable de configurer et maintenir automatiquement les règles d'un ou plusieurs projets GitLab.

## Contraintes techniques
GitLab, API REST GitLab, GitLab CI/CD

## Fonctionnalités
Gérer les branches protégées, gérer les règles de Merge Request, gérer les règles d'approbation, vérifier la conformité d'un dépôt, réappliquer les règles si nécessaire.

## Attendu
Le projet doit être simple, modulaire et facilement extensible.
